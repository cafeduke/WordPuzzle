How to use the tool
-------------------
1) Install JDK

Usage
-----
Puzzle

Execute Command
---------------
1) Windows+R (Run)
2) Type "cmd" > Enter
3) cd D:\Java\WordPuzzle
4) D:
5) Puzzle -row 15 -col 15 -word grape,watermelon,apple,lemon,orange,mango,jackfruit,pineapple -file MyFruit.html -title "My Fruit"

Generate Multiple Puzzles
-------------------------
1) Windows+R (Run)
2) Type "cmd" > Enter
3) cd D:\Java\WordPuzzle
4) D:
5) start notepad MultiplePuzzles.cmd > Save
6) MultiplePuzzles
